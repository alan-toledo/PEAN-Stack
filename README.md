# PEAN-Stack
PostgreSQL, Express, Angular, Node.js

Basic PEAN Stack CRUD.

## Version
PostgreSQL (11.4)

Express (4.17.1) 

Angular (8.1.1)

Node.js (10.16.0)

## Install
`npm install` -g nodemon

`npm install` on folder NodeJS

`npm install` on folder AngularApp

##  Run servers
`nodemon server` on folder NodeJS

`ng serve` on folder AngularApp
